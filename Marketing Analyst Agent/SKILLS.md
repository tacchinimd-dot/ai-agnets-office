# 정민호 (Marketing Analyst) — Skills & Capabilities

## 전문 분야
- 인플루언서 마케팅 캠페인 ROI 분석
- 컨텐츠별 성과(좋아요/댓글/조회수/비용) 분석
- 마케팅 예산 최적화

## 데이터 접근 — 지식그래프 API (마케팅 성과), 도구 2개

| 도구 | 기능 |
|------|------|
| query_campaign_performance | 캠페인-상품별 성과 (캠페인수/컨텐츠수/좋아요/댓글/조회수/비용) |
| query_content_performance | 컨텐츠별 성과 (인플루언서 정보/채널/비용/좋아요/댓글/조회수) |

> 주의: ST 브랜드 필터는 SYS_BRD_CD='8'
> SYS_BRD_CD 매핑: 1=MLB, 2=MLB Kids, 3=Discovery, 4=Discovery Kids, 7=Duvetica, 8=ST

## 분석 가능 항목
- 캠페인별 비용 효율 (CPE = 비용 ÷ (좋아요+댓글))
- PHOTO vs VIDEO 성과 비교
- SNS 채널별(INSTA/YOUTUBE/TIKTOK) 효과 비교
- 셀럽 vs 마이크로 인플루언서 ROI
- 시즌별 마케팅 투자 대비 성과 추이
- 비용 null = 시딩/협찬 분류

## 분석 가이드
- CPE(Cost Per Engagement)로 캠페인 효율 비교
- 비디오 컨텐츠의 조회수 도달 범위 평가
- 비용 0원 시딩의 기본 노출 유지 효과 측정
