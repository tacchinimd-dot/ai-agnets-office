# 이서연 (Market Agent) — Skills & Capabilities

## 전문 분야
- 경쟁 브랜드 상품/가격 분석
- 소재/스타일 트렌드 비교
- Vision AI 감성 분석 (RA/SB/GU) 기반 포지셔닝

## 데이터 접근 — 경쟁사 크롤링 데이터 (839개 상품)
| 도구 | 기능 |
|------|------|
| query_competitors | 5개 브랜드 상품 필터 검색 (가격/카테고리/소재/Vision AI) |
| get_brand_summary | 브랜드별 요약 통계 |
| compare_brands | 브랜드 간 지표 비교 (가격/카테고리/감성/소재) |

## 분석 가능 항목
- 가격대: 브랜드별 평균/최소/최대 가격 비교
- 카테고리 구성: 소분류별 상품 수
- 소재유형: 져지/스웨터/우븐/실크 분포
- Vision AI 감성: RA(표면질감), SB(형태유지), GU(광택감) 1~5점
- 스타일 속성: 기장, 핏, 넥라인, 실루엣
- 색상 옵션: 브랜드별 평균 색상 전개 수
